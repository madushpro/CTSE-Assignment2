INITIALIZATION PATTERN (ALL AGENTS)
================================================================================

```python
def __init__(self, use_ollama: bool = True):
    # ... agent setup ...
    
    self.ollama_client: Optional[OllamaClient] = None
    self.use_ollama = use_ollama and OLLAMA_AVAILABLE
    
    if self.use_ollama:
        try:
            self.ollama_client = create_ollama_client()
            logger.info("✓ Ollama LLM initialized for [Agent Name]")
        except Exception as e:
            logger.warning(f"Could not initialize Ollama: {str(e)}. Using mock data.")
            self.use_ollama = False
```

Benefits:
✓ Graceful degradation (works without Ollama)
✓ Clear logging of initialization status
✓ Automatic fallback to mock data
✓ No hard failure if Ollama unavailable
